Toys

domain: https://toys-api-18bf.onrender.com/

Some "GET" requests you can get information about the API, By adding these queries you can see the edit

  • Per-Page : you can select the amount of obj per one page.
  • Page : you can select the num of pages you want.

All toys

GET: Get a list of all the toys, will present 10 results per page.

10 toys on page 1:

/toys/all?page=1

30 toys on page :

/toys/all?perPage=30

Search by name/info

GET: search for a toy by searching follwing name or info after the "s=", it will present 10 results per page.

/toys/search?s={name or info}
/toys/search?s=car

Get by price

GET: Get a list of the toys by min & max , it will present 10 results per page.

/toys/price?min={min price}&max={max price}
/toys/price?min=35&max=60

Get by gategory

GET: Get a list of all the toys in toys category , it will present 10 results per page.

/toys/category/{name category}
/toys/category/strategy

Get by id

GET: Get the details of a toy by ID

/toys/single/{id toy}
/toys/single/647c38297bb0d9ed415e2922

Add new toys

POST: Add a new toy to the data base, You need a token for this query (if you want to see the template of valid toy) 

/toys

 {
    "name":"lolo doll",
    "info":"A funny doll that children love",
    "category":"Doll",
    "img_url":"lolo.jpg" ,
    "price":40
}

Update toy

PUT: Update a toy in the data base, You need a token for this query

/toys/{id}

 {
    "name":"lolo love doll",
    "info":"A funny doll that children love",
    "category":"Doll",
    "img_url":"lolo.jpg" ,
    "price":40
}

Delete toy

DELETE: Delete a toy from the data base, You need a token for this query

/toys/{id}

 {
    "msg": "The toy has been successfully deleted"
}
