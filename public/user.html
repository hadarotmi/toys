Users

User list

GET: Get list of all users - by token admin .

http://localhost:3002/users/all

My user info

GET: Get a list of all the toys, will present 10 results per page.

http://localhost:3002/users/myInfo

Delete user

DELETE: Delete a user from the data base. Admin token can delete every user , and user can delete only himself.

http://localhost:3002/user/{id}

Add new user

POST: Add a new user to the data base, You need a token for this query

http://localhost:3002/user/

Update user

PUT: Update your user in the data base, Admin token can edit every user data , and user can edit only himself.

http://localhost:3002/user/{id}
