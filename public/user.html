User list

GET: Get list of all users - by token admin .

/users/all

My user info

GET: Get a list of all the toys, will present 10 results per page.

/users/myInfo

Delete user

DELETE: Delete a user from the data base. Admin token can delete every user , and user can delete only himself.

/users/{id}

Add new user

POST: Add a new user to the data base, You need a token for this query (if you want to see the template of valid user) 

/users

 {
    "name": "jack black",
    "email": "jack12@walla.com",
    "pass":"123jack"
}

Login user

POST: User login, if the email and password are correct the server will respond with a token,
(if you want to see the template of valid user) 

/users/login

{
    "email": "jack12@walla.com",
    "password": "123jack"
}

Update user

PUT: Update your user in the data base, Admin token can edit every user data , and user can edit only himself.

/users/{id}
